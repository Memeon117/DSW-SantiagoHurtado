Etiquetas semánticas en HTML/HTML5

¿Qué es la semántica?

La semántica es el estudio del significado de las palabras y frases en el lenguaje. Por consecuente tenemos que Elementos Semánticos=Elementos con un significado.

¿Qué son los elementos o etiquetas semánticas?

Un elemento semántico claramente describe su significado para ambos, el navegador y el desarrollador web. Se les conoce como Elemento o Etiqueta. Para poder comprender mejor un ejemplo de una etiqueta semántica empezaremos dando ejemplo todo lo contrario, una etiqueta no semántica:

Ejemplo de etiquetas no-semánticas: div y span, estos no dicen nada al respecto de su contenido

Ejemplo de etiquetas semánticas: form, table, article, definen claramente la clase de contenido dentro de los mismos.

Etiqueta SECTION

Este se utiliza para representar una sección general dentro de un documento o una aplicación, algo similar al capítulo de un libro. Puede contener subsecciones, lo podemos acompañar con h1-h6 y así estructurar mejor toda la página, esto le dará una jerarquía al contenido que se quiera plasmar en la web

Para ver la aplicación del pseudocodigo HTML5 de la etiqueta SECTION consultar el capítulo 2 "SEMÁNTICA" de HTML5 ARTKAITZ GARRO, en la sección 2.4.5 "SECTION".

Etiqueta ASIDE

Representa una sección de la poágina que abarca contenido relacionado con el contenido que lo rodea, por lo que es un contenido independiente. Este elemento puede utilizarse para barras laterales, elementos publicitarios, para grupos de elementos de la navegación u otro contenido que se considere separado del principal de la página

Para ver la aplicación del pseudocodigo HTML5 de la etiqueta ASIDE, Consultar el Capítulo 2 "SEMÁNTICA" de HTML5 ARKAITZ GARRO, en la sección 2.4.6 "ASIDE".

Etiqueta FOOTER

Representa el pie de una sección, con información acerca se la página, este contenido no está muy relacionado con lo principal de la página, allí se puede plasmar información legal, el autor, el copyright o el año

Para ver la aplicación del pseudocodigo HTML5 de la etiqueta, consultar el Capítulo 2 "SEMÁNTICA" de HTML5 ARKAITZ GARRO, en la sección 2.4.3 "FOOTER".

Etiqueta ARTICLE

Representa un compuesto autónomo de la página, aplicación o sitio web, tiene como intención ser reutilizado y repetido. Podría utilizarse en los artículos de foros, revistas o un artículo de un periódico, la entrada a un blog, comentarios escritos por los usuarios, Entre otros.

Cuando estos elementos o etiquetas son anidados, los elementos interiores representan los artículos que en principio son relacionados con el contenido del artículo externo. Por ejemplo, un artículo de un blog permite comentarios de usuario, dichos comentarios se pueden representar con article.

Para ver la aplicación del pseudocodigo HTML5 de la etiqueta ARTICLE, consultar el Capítulo 2 "SEMÁNTICA" de HTML5 ARKAITZ GARRO, en la sección 2.4.4 "ARTICLE".
